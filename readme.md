# Hospital Patient & Revenue Analytics

## Project Overview

This project analyzes hospital patient and admission data using Python, SQLite, Pandas, and Matplotlib. The goal is to identify patient trends, revenue patterns, disease statistics, and hospital performance metrics through SQL-based analysis and data visualization.

## Technologies Used

* Python
* Pandas
* SQLite
* Matplotlib
* SQL

## Dataset

The project uses two datasets:

### Patients Table

Contains:

* Patient ID
* Patient Name
* Age
* Gender
* City

### Admission Table

Contains:

* Admission ID
* Patient ID
* Disease
* Doctor
* Department
* Admission Date
* Discharge Date
* Bill Amount

## Analysis Performed

### Patient Analysis

* Total number of patients
* City-wise patient distribution
* Patients with multiple admissions
* Disease-wise patient count

### Revenue Analysis

* Revenue by department
* Revenue by doctor
* Revenue by disease
* Patient ranking based on total hospital spending
* Running revenue trend over time

### Hospital Stay Analysis

* Average hospital stay duration
* Disease-wise average stay duration

### Advanced SQL Concepts Used

* GROUP BY
* HAVING
* JOINS
* Window Functions

  * RANK()
  * SUM() OVER()
* Date Functions

  * JULIANDAY()

## Key Insights

* Identified departments generating the highest revenue.
* Determined diseases associated with longer hospital stays.
* Ranked patients based on total spending.
* Analyzed cumulative hospital revenue trends.
* Evaluated doctor-wise revenue contributions.

## Visualizations

The project includes:

* Revenue by Department
* Revenue by Doctor
* City-wise Patient Distribution
* Running Revenue Trend
* Average Stay Duration by Disease
